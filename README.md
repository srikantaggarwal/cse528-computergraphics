CSE528 Computer Graphics "Meshless Deformations Based on Shape Matching"

Implemented "Meshless Deformations Based on Shape Matching", a SIGGRAPH 2005 paper by M. Muller, B. Heidelberger, M. Teschner and M. Gross as a part of the Computer Graphics course (CSE 528). The implementation was done using OpenGL, C++ and Eigen Library.
